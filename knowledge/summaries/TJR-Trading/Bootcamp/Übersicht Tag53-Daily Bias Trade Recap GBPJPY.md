### Der Trade: GBP/JPY Short – Vollständige Execution

**Schritt-für-Schritt:**
1. Täglich bärisch + täglicher OB identifiziert und getestet ✓
2. London Session Hochs und Asian Session Hochs als Liquidität markiert ✓
3. Kurs sweept BEIDE Hochs (London + Asian) → Liquiditätssweep ✓
4. 5-Minuten Strukturbruch nach unten ✓
5. Einstieg vom 1-Minuten-Chart für präzisen Entry ✓
6. Take Profit 1: Order Block darunter → getroffen ✓
7. Nach TP1: 50% der Position geschlossen ✓
8. Stop auf Break-Even gesetzt ✓
9. Verbleibende Position: +~$13.000 floating

### Besondere Lernpunkte

- **Liquiditätssweep beider Sessions:** London Hoch UND Asian Hoch werden beide genommen → stärkeres Signal
- **1-Minuten-Einstieg:** Nach 5m BoS auf 1m für präzisen Entry wechseln → kleinerer Stop
- **TP1 schließen + BE-Stop:** Klassisches Risk-Management nach dem System

### Bias-Bestätigung vor Trade

- GBP/JPY täglich bärisch → Short-Bias bestätigt
- Wöchentlich bärisch → wöchentliches Retracement läuft
- Daily OB reagiert → Institutionen verkaufen hier
- Multi-TF Alignment: Weekly + Daily bärisch → maximale Confluence für Short
