# Zusammenfassung: Aktienkurs bewerten
**Erstellt:** 2026-06-06 | **Quelle:** knowledge/videos/finanzfluss/Aktienkurs bewerten/

---

## 1. Grundlagen Aktien & Börse

- **Aktie** = Anteil an einem Unternehmen; Aktionäre sind Miteigentümer
- **Rendite** kommt aus Kursgewinnen + Dividenden
- Börse ist kein Kasino – langfristig spiegelt der Kurs den Unternehmenswert wider
- **Sparplan-Prinzip**: Regelmäßig kaufen → Cost-Averaging-Effekt nutzen

---

## 2. Die 12 wichtigsten Aktienkennzahlen

| Kennzahl | Bedeutung | Richtwert |
|----------|-----------|-----------|
| **KGV** (Kurs-Gewinn-Verhältnis) | Preis / Gewinn je Aktie | ~15 = fair, >25 = teuer |
| **KBV** (Kurs-Buchwert) | Preis / Eigenkapital | <1 = günstig |
| **KUV** (Kurs-Umsatz) | Preis / Umsatz | Branchenabhängig |
| **EPS** (Earnings per Share) | Gewinn je Aktie | Wachstum = positiv |
| **Dividendenrendite** | Dividende / Kurs | 2–4% = solide |
| **Payout Ratio** | Dividende / Gewinn | <60% = nachhaltig |
| **EV/EBITDA** | Enterprise Value / Betriebsgewinn | ~10 = fair |
| **Free Cashflow** | Barer Gewinn nach Investitionen | Positiv = stark |
| **Eigenkapitalquote** | EK / Gesamtkapital | >30% bevorzugt |
| **Verschuldungsgrad** | Fremdkapital / Eigenkapital | <100% besser |
| **ROIC** | Return on Invested Capital | >10% = stark |
| **Gewinnwachstum** | YoY Gewinnsteigerung | >10% = Wachstum |

---

## 3. Aktien richtig bewerten – 2 einfache Methoden

### Methode 1: KGV-Vergleich
- Aktuelles KGV vs. historisches KGV des Unternehmens
- Aktuelles KGV vs. Branchendurchschnitt
- **Faustformel**: KGV < Gewinnwachstum (PEG-Ratio < 1 = günstig)

### Methode 2: Discounted Cashflow (DCF)
- Zukünftige Cashflows schätzen und auf heute abdiskontieren
- Einfach: Intrinsic Value = EPS × erwartetes KGV in 5 Jahren
- Sicherheitsmarge von 20–30% einplanen

---

## 4. Chartanalyse – Hilfreich oder nicht?

**Pro:**
- Visualisiert Momentum und Trendstärke
- Unterstützungs-/Widerstandszonen sichtbar
- Psychologische Niveaus (runde Zahlen, All-Time-Highs)

**Contra:**
- Keine Vorhersagekraft bei Fundamentaldaten
- Self-Fulfilling Prophecy – funktioniert nur, weil alle drauf schauen
- Für Buy-and-Hold-Investor wenig relevant

**Fazit:** Chartanalyse als Ergänzung zur Fundamentalanalyse sinnvoll, nicht als Ersatz.

## 5. Denkfehler beim Investieren (Behavioral Finance)

| Bias | Beschreibung | Gegenmittel |
|------|-------------|-------------|
| **Bestätigungsfehler** | Nur Infos suchen, die eigene Meinung bestätigen | Aktiv Gegenargumente suchen |
| **Verlustaversion** | Verluste schmerzen 2× stärker als gleiche Gewinne | Langfristperspektive einnehmen |
| **Ankering** | Kaufkurs als Referenzpunkt fixiert | Fragen: Würde ich heute kaufen? |
| **Herdentrieb** | Kaufen wenn alle kaufen, verkaufen wenn alle verkaufen | Eigenständige Analyse |
| **Overconfidence** | Eigene Prognosen überschätzen | Historische Hit-Rate prüfen |
| **Recency Bias** | Jüngste Entwicklung wird zu stark gewichtet | 10+-Jahresbetrachtung |
| **Sunk Cost** | Verlustposition halten "weil schon viel investiert" | Entscheidung unabhängig vom Einstandskurs |

## Zusammenfassung (3 Punkte)

- **KGV + EV/EBITDA + Free Cashflow** sind die wichtigsten Bewertungskennzahlen – immer im Branchenvergleich betrachten
- **Chartanalyse** hat Nutzen für Einstiegszeitpunkt, ersetzt aber keine Fundamentalanalyse
- **Behavioral Biases** sind die größte Gefahr für Privatanleger – systemisches Vorgehen (Checkliste, feste Strategie) schützt davor
